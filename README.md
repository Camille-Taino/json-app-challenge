### Running the App
1. Open the project folder in your favorite code editor.
2. Open the terminal/ctrl + `.
3. On the terminal run'npm install"
4. Then 'npm start'.
5. Copy the localhost url (sample: "http://localhost:3000/") then paste it on the chrome browser.
